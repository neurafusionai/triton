# Triton Avatar Concepts (Prototype)

This folder contains a code-generated SVG prototype for the **Tensor Wave** avatar concept:

- `tensor-wave-avatar.svg`: A dark-mode-friendly mark combining a wave silhouette, thread-block grid motifs, and a glowing `T` crest.

## Regenerate

```bash
python3 scripts/generate_tensor_wave_avatar.py
```

Optional flags:

- `--size 1024` to generate a larger canvas.
- `--output docs/branding/custom-name.svg` to write to a different path.
